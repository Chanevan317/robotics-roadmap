<Card title="Little Introduction">
    Welcome to the roadmap for modern robotics. As the world shifts toward smart, AI-driven machines, this course takes you from an absolute beginner to a capable robotics developer. You will progress through these simple steps: mastering the fundamental concepts, learning softwares and tools, building virtual worlds in Isaac Sim, training intelligent behaviors with Reinforcement Learning, connecting code to real hardware via ROS 2, and exploring cutting-edge AI that lets robots see and understand human commands.
</Card>

<Aside>
  <strong>Prerequisites:</strong> A basic understanding of programming (Python and introductory C++) and foundational mathematics (Linear Algebra, Calculus, and basic Probability).<br />
  <strong>Course Format:</strong> Self-paced and written like real project documentation. We designed it this way as learning through hands-on docs is exactly how real engineers pick up new skills and learn.
</Aside>
